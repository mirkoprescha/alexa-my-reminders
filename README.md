# alexa-my-reminders

## dependencies
This skill uses https://github.com/anjishnu/ask-alexa-pykit
```
mkdir alexa-my-reminders
cd alexa-my-reminders
pip install ask-alexa-pykit --target .
```

create local Venv for boto3 (if needed)
```
cd alexa-my-reminders
virtualenv venv
source venv/bin/activate

pip install boto3
```
